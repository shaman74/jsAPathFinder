Path finder


linear movement cost (lc)
diagonal movement cost (dc)

Heuristic function

manhattan h(x)=min(lc,dc)*(abs(x-xt)+abs(y-yt))
diagonal h(x)=dc*max(abs(x-xt),abs(y-yt))
euclidean h(x)=dc*sqrt(sqr(x-xt)+sqr(y-yt))
euclidean^2 h(x)=dc*(sqr(x-xt)+sqr(y-yt))
